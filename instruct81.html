Install Windows 8.1

It's Basically The Same Thing As Windows 10

I Would Recomend The Media Creation Tool

Download A Media Creation Tool For Windows 8.1 By Google Search

Get An 4 GB USB Or A 4 GB CD Or SD

Then, Plug It In And Start The Windows 8.1 Media Creation Tool

Select The Version Of The Computer (Most Likely: 64-bit) And Countinue

Then, Click USB If You Have A USB Or ISO File If You Have The Other

For SD Or DVD Users, Copy The ISO File To The SD Or DVD And Follow Along

Wait For It To Install, Then Reboot Your Computer And On Startup, Press F8 Or F12

Use Arrow Keys To Select Your USB, Or SD Or DVD And Enter To Select

It Will Boot, Probobly

Make Sure You Have Your Product Key Ready

Also, You Don't Need To Do All That, But You Might Need To,

Worth A Shot, Double-Click The ISO File And If A Menu Opens That Has Files In It, Then Well Done

Double-Click The File Called "setup.exe" And Follow Along The Setup

You Will Need A Product Key
